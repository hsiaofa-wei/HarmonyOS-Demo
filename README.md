# HarmonyOS 智能对话客户端 Demo

> HarmonyOS / ArkTS / ArkUI / 通义千问 API / Prompt Engineering / AI Chat Demo

基于 HarmonyOS（ArkTS + ArkUI）开发的 AI 智能对话客户端 Demo，接入通义千问大模型 API，实现基础聊天界面、消息发送、AI 回复、Prompt 角色设定与状态提示，聚焦移动端 AI 对话场景下的交互体验与功能落地。

---

## 项目简介

本项目是一个面向移动端场景的 HarmonyOS AI 对话 Demo，使用 ArkTS + ArkUI 完成客户端页面开发，并通过网络请求接入通义千问大模型 API，实现用户与 AI 的基础对话能力。

项目当前以 **“轻量可运行 Demo”** 为目标，重点完成：

- 聊天界面搭建
- 用户消息发送
- 大模型接口调用
- AI 回复展示
- Prompt 角色设定
- 加载状态 / 异常提示
- API Key 未配置提醒

---

## 项目亮点

- 基于 **HarmonyOS + ArkTS + ArkUI** 实现移动端 AI 对话客户端
- 接入 **通义千问 API**，完成用户输入到模型回复的完整调用链路
- 支持 **Prompt 角色设定**，可通过预设角色优化模型回复风格与场景表现
- 实现基础聊天交互体验，包括：
  - 消息列表展示
  - 发送中提示
  - AI 思考中提示
  - API Key 未配置提醒

---

## 技术栈

- **HarmonyOS**
- **ArkTS**
- **ArkUI**
- **HTTP 网络请求**
- **通义千问 API（DashScope / Qwen）**
- **Prompt Engineering**
